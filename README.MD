# ***DevLobby*** Web application
 - A social network for developers.
 - Build with Django/python.
 - Django learning project.
 
 # Function
 - Login / Register (OAuth and integrate fb/google )
 - Create blog posts
 - Post status update
 - Message each other
 - Notification
 - Search function
